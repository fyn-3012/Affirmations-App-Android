# Affirmations App

Aplikasi Android yang menampilkan daftar kartu afirmasi 
(kata-kata motivasi) yang dapat di-scroll, dibangun 
menggunakan Jetpack Compose.

## Fitur
- Menampilkan 10 kartu afirmasi dengan gambar dan teks
- Daftar dapat di-scroll menggunakan LazyColumn
- Tampilan kartu menggunakan komponen Card Material3

## Teknologi
- Kotlin
- Jetpack Compose
- LazyColumn
- Material3 Card
